모든 시각적 디자인은 추후 변경될 수 있습니다 
가장 간단한 인터페이스만 구현해주시면 됩니다. 
페이지는 3개 입니다 
이미지 파일을 직접 만들까 하다가 HTML 생성을 부탁했습니다.

1.메인 페이지 
eq, 변환 각 페이지로 이동하는 버튼 
상단 로고 클릭 시 메인페이지로 이동 (모든페이지 동일) 
좌상단 메뉴는 추가를 고민중인 페이지로 이동(현재는 의미없음)


EQ 페이지로 이동 시 파일 업로드창이 뜨게 됩니다 
허용 규격은 모든 음악파일 (AAC,mp3,flac,wav)중 길이가 10분보다 짧은 것
모든 파일은 10mb이하가 될 때까지(또는 서버 부담이 적을 선까지) mp3로 압축합니다
압축은 크기에 따라서 192kbps까지 압축하나 그래도 부족하다면 경고메세지를 띄웁니다

또는 기본음원으로 시작하기(기본 서버 음원)으로 바로 eq창으로 넘어갈 수 있습니다 

2. eq 조절 페이지 

오른쪽 필터가 그래픽적으로 표기되는 부분의 그리드는 AI가 잘 표현하지 못하여 이미지를 첨부합니다 (GRID.png)
GEQ PEQ를 선택 가능하며 자세한 알고리즘은 제가 직접 구현합니다 
두 종류의 eq는 서로에게 영항을 주지 않으며 
최상단의 음원 재생 기능은 지금 선택된 EQ의 필터를 기반으로 ON OFF 를 걸어 재생이 가능하게 됩니다

GEQ 
 31개의 미리 설정된 주파수 영역별로 -15~15dB까지 게인 컨트롤이 가능합니다
PEQ 
 무한정으로 필터를 추가할 수 있으며 
왼쪽엔 주파수 / 게인(dB) /Q /필터종류 의 4가지 정보가 표기됩니다 
오른쪽 그리드에 존재하는 숫자 아이콘을 클릭하면 각 필터에 맞는 창이 뜨며 위치는 아이콘과 인접한 우상단 이어야 합니다. 
창에는 필터 유형 주파수 게인 Q 를 직접 입력할 수 있으며 
마우스 드래그를 통해 상하는 게인 좌우는 주파수 그리고 휠조작을 통해 q값을 줄이거나 늘릴 수 있습니다 

마지막으로 가장 하단에 내보내기 버튼을 TXT파일로 내보내집니다(규격은 추후 설정)
